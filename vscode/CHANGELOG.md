# Changelog

## 1.0.0 - Initial Release

### Features
- Syntax highlighting for Ado language
- Code snippets for common patterns
- Language configuration for comments and brackets

### Supported Syntax
- Array slicing (`arr[2..5]`)
- List comprehensions (`[for i in 1..10 i*i]`)
- Destructuring (`let [a, b, ...rest] = arr`)
- All core language keywords and operators
